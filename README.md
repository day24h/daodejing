### 编辑、编译 pdf 文件：

1. **注册**：https://typst.app/ 免费账户功能够用。零安装，网页编辑、编译，手机上也能操作。
2. **新建项目**：在 Dashboard 页面，点击 Empty document 大按钮。
3. **上传代码**：菜单栏 → File → Upload file，将本地 `编译用代码/` 里的所有内容，上传至项目。
4. **指定预览文件**：在 Files 侧栏中 找到 `main.typ` 文件，右键将其设置为预览文件。
5. **完成**：导出 pdf。
    * 修改代码，可以参考 `代码说明.pdf`。
    * 本 Typst 文档项目的分享地址（只读）：https://typst.app/project/rh30kTYz7JlHgiY2e0m3T4
